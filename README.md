Unity Cache Server
==================

.. in Go

Installation from source
========================

    go install github.com/msiebuhr/ucs/cmd/ucs
    ucs -h

There's also a quick-and-dirty loadtest utility, `ucs-bender`:

    go install github.com/msiebuhr/ucs/cmd/ucs-bender
    ucs-bender
